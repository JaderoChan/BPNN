# 函数对比与使用推荐

本文档描述了代码中包含的各激活函数，并给出了他们与损失函数的常见搭配和适用场景。

## 激活函数对比

| 激活函数 | 公式 | 优点 | 缺点 | 适用层 | 备注 |
| ------ | --- | ---- | --- | ----- | ---- |
| Sigmoid | $f(x)=\frac{1}{1+e^{-x}}$ | 平滑；有概率解释 | 存在梯度消失问题；输出非零中心 | 隐藏层/输出层 | 输出范围为 $(0,1)$。适合用于二分类、多标签分类的输出层。 |
| Tanh | $f(x)=\frac{e^x-e^{-x}}{e^x+e^{-x}}$ | 平滑；零中心 | 存在梯度消失问题；无概率解释 | 隐藏层 | 输出范围为 $(-1,1)$。一般不用于输出层，除非用于回归任务且输出在 $(-1,1)$ 范围内。 |
| ReLU | $f(x)=max(0,x)$ | 计算快；缓解梯度消失问题 | 神经元可能“死亡”（输出永久为 0） | 隐藏层 | 输出范围为 $[0,+\infty)$。几乎只用于隐藏层。 |
| Leaky ReLU | $f(x)=max(\alpha x,x)$ | 解决了 ReLU 的神经元死亡问题 | 多一个超参数 $\alpha$；实践中与 ReLU 的差异不大 | 隐藏层 | 输出范围为 $[0,+\infty)$。几乎只用于隐藏层。 |
| Softmax | $f(x)=\frac{e^{z_k}}{\sum_{j=1}^Ke^{z_j}}$ | - | - | 输出层 | 输出范围为 $(0,1)$，且和为 $1$。适合用于单标签多分类的输出层。将任意实数向量压缩成概率分布。（$K$ 为输出节点数量，$z_k$ 为第 $k$ 个输出节点激活前的值） |
| Linear | $f(x)=x$ | - | - | 输出层 | 线性函数（无激活），输出范围为 $[0,+\infty]$。适合用于一般回归任务的输出层。 |

## 任务搭配

| 任务类型 | 适用场景 | 输出层激活函数 | 损失函数 |
| ------ | ---- | ------------ | ------ |
| 二分类 | 判断是非关系 | Sigmoid（一个输出节点） | 二元交叉熵函数 BCE |
| 单标签多分类 | 单标签/互斥分类场景（如数字识别） | Softmax | 多分类交叉熵函数 CCE |
| 多标签多分类 | 多标签分类场景（如图片分类场景） | Sigmoid （每类一个输出节点） | 二元交叉熵函数 BCE |
| 回归任务 | 拟合曲线 | Linear | 均值误差 MSE |
